# VL.M8Display

[Dirtywave](https://dirtywave.com) M8 Remote Display for [vvvv gamma](https://visualprogramming.net/)

## Installing

Grey menu (top left) -> Manage Nugets -> Commandline

```
nuget install VL.M8Display
```
